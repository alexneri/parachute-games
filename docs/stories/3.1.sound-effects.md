# Story 3.1 — Sound Effects

**Epic**: 3 — Audio  
**Status**: Draft  
**Priority**: P1

## User Story
As a player, I want to hear synthesized catch, miss, and game-over sounds so that the game has the right feel.

## Acceptance Criteria
- [ ] `playCatch()`: 880Hz square wave, 80ms duration
- [ ] `playMiss()`: 120Hz sawtooth wave, 300ms, pitch decays to 80Hz
- [ ] `playGameOver()`: descending arpegio 880→660→440→220Hz, ~1s total
- [ ] All sounds synthesized via Web Audio API OscillatorNode (no audio files)
- [ ] AudioContext initialized lazily on first user interaction (not on page load)
- [ ] Muted state silences all sounds without throwing errors

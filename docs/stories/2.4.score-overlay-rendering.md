# Story 2.4 — Score & Overlay Rendering

**Epic**: 2 — LCD Presentation Layer  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want the score to display in a 7-segment LCD style and overlays (MISS, GAME OVER) to render correctly so that game state is always readable.

## Acceptance Criteria
- [ ] Score 0–999 renders correctly as three 7-segment digits
- [ ] Off-segments visible at ~5% opacity
- [ ] MISS text renders at (290, 238) in miss_flash phase only
- [ ] Shark renders at miss X position in miss_flash phase
- [ ] GAME OVER overlay: semi-transparent dark tint + centered text
- [ ] PUSH START text flashes at 1Hz in attract and game_over phases

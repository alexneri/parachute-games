# Story 1.6 — Score System

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want my score to increment on each catch so that I have a goal to pursue.

## Acceptance Criteria
- [ ] Score starts at 0 on new game
- [ ] Score increments by 1 per catch event
- [ ] Score cannot exceed 999 (wraps to 000 if it would)
- [ ] Score resets to 0 on restart

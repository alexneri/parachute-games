# Story 2.2 — Sprite System

**Epic**: 2 — LCD Presentation Layer  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want all game sprites to look like the original Game & Watch silhouettes so that the game feels authentic.

## Acceptance Criteria
- [ ] All 11 sprite keys have vector draw functions: helicopter, parachutist_zone0/1/2, boat, boat_catching, shark, water_1/2, palm_left, palm_right
- [ ] Sprites render at correct canvas positions per constants
- [ ] Sprite color is #1a1a1a (not pure black)
- [ ] Static sprites (palms, water) are pre-rendered to offscreen canvas for performance
- [ ] No sprite draws outside its documented bounding box

# Story 2.3 — LCD Ghost Effect

**Epic**: 2 — LCD Presentation Layer  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want previous sprite positions to linger at low opacity so that the game looks like real LCD hardware.

## Acceptance Criteria
- [ ] Moving boat creates ghost at previous position
- [ ] Ghost initial opacity: 0.12 (GHOST_OPACITY constant)
- [ ] Ghost decays linearly to 0 over 100ms (GHOST_DURATION_MS)
- [ ] Expired ghosts are pruned from the ghost list each frame
- [ ] Effect disabled when prefers-reduced-motion is active
- [ ] Ghost list never exceeds 20 entries (safety cap)

# Story 4.1 — Game A / B Modes

**Epic**: 4 — Game Modes & Polish  
**Status**: Draft  
**Priority**: P1

## User Story
As a player, I want to choose between Game A and Game B so that I can play at the difficulty that suits me.

## Acceptance Criteria
- [ ] Game A: standard speed per documented timing constants
- [ ] Game B: fall rates and helicopter traversal ~1.3× faster than Game A
- [ ] Game B: max 3 simultaneous parachutists (vs. 2 for Game A)
- [ ] Mode selector buttons on device frame reflect active mode visually
- [ ] Mode change only applies when in 'attract' or 'game_over' phase
- [ ] Mode change during 'playing' is silently ignored

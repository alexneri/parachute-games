# Story 1.1 — Game Loop Infrastructure

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a developer, I want a 60fps requestAnimationFrame game loop with fixed-timestep physics updates so that game logic is deterministic and rendering is smooth.

## Acceptance Criteria
- [ ] Loop runs at 60fps via `requestAnimationFrame`
- [ ] Physics updates at fixed timestep (GAME_A_TICK_MS or GAME_B_TICK_MS)
- [ ] Fixed-timestep accumulator pattern prevents spiral of death
- [ ] Loop pauses automatically when tab is hidden (`visibilitychange` event)
- [ ] Loop resumes when tab becomes visible again
- [ ] delta capped at 250ms max to prevent catch-up burst after tab switch

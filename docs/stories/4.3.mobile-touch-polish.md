# Story 4.3 — Mobile & Touch Polish

**Epic**: 4 — Game Modes & Polish  
**Status**: Draft  
**Priority**: P1

## User Story
As a mobile player, I want reliable touch input without accidental scrolling so that the game plays well on phone.

## Acceptance Criteria
- [ ] `touchstart` event handler calls `preventDefault()` to stop page scroll
- [ ] Tap left half of canvas (x < canvasWidth/2) calls `engine.moveBoat('left')`
- [ ] Tap right half calls `engine.moveBoat('right')`
- [ ] Tap center area (30–70% of canvas width) starts/restarts game
- [ ] Event listeners set with `{ passive: false }` option
- [ ] No double-tap zoom on the canvas element
- [ ] Visual touch zones tested on iOS Safari 16+

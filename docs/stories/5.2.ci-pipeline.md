## User Story
As a developer, I want lint, type-check, unit tests, and build checks on every PR so that regressions are caught automatically.

## Acceptance Criteria
- [ ] GitHub Actions workflow runs on push and pull_request
- [ ] `npm run type-check` (tsc --noEmit) must pass
- [ ] `npm run lint` (eslint) must pass
- [ ] `npm run test` (vitest run) must pass with 0 failures
- [ ] `npm run build` (next build) must succeed
- [ ] Lighthouse CI checks on preview URL: Performance ≥ 90, Accessibility ≥ 85
- [ ] Failed checks block merge via branch protection rules

# Story 3.2 — Audio Controls

**Epic**: 3 — Audio  
**Status**: Draft  
**Priority**: P1

## User Story
As a player, I want to mute/unmute the game so that I can play silently in public.

## Acceptance Criteria
- [ ] Mute toggle button visible in device frame controls row
- [ ] Mute icon (speaker with waves) shown when unmuted
- [ ] Muted icon (speaker with slash) shown when muted
- [ ] Mute state persists across game rounds (not reset on restart)
- [ ] `aria-pressed` attribute reflects mute state
- [ ] AudioContext not created until first user interaction regardless of mute state

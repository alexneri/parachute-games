# Story 1.2 — Helicopter Entity

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want to see a helicopter traverse from right to left and drop parachutists so that gameplay can begin.

## Acceptance Criteria
- [ ] Helicopter spawns at right edge (stepIndex = HELICOPTER_STEPS)
- [ ] Moves left in discrete steps per game tick
- [ ] Drops parachutist at step 7 and step 3 (if under max count)
- [ ] After full traverse, enters cooldown before respawning
- [ ] Cooldown: GAME_A_HELI_COOL_MS (mode A), GAME_B_HELI_COOL_MS (mode B)
- [ ] Multiple parachutists cannot exceed MAX_PARA limit per mode

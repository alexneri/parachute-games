**Epic**: 2 — LCD Presentation Layer  
**Status**: Draft

## User Story
As a developer, I want a Canvas 2D rendering system with the correct LCD background and sprite pipeline so that all game entities can be drawn faithfully.

## Acceptance Criteria
- [ ] Canvas renders LCD background color #b8c9a3 on first frame
- [ ] Canvas scales correctly at 1×, 2× (Retina), and 3× device pixel ratio
- [ ] rAF loop calls engine.tick and renderer.render each frame
- [ ] ResizeObserver recalculates scale without corrupting game state
- [ ] Canvas element has correct aria-label attribute
- [ ] Canvas maintains 4:3 aspect ratio at all viewport sizes

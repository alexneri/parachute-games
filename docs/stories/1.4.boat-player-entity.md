# Story 1.4 — Boat Player Entity

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want to move my boat left and right to catch falling parachutists so that I can play the game.

## Acceptance Criteria
- [ ] Boat has exactly 5 discrete X positions (indices 0–4)
- [ ] Each input moves boat one position in the given direction
- [ ] Boat cannot move below position 0 or above position 4
- [ ] Input is ignored when game phase is not 'playing'
- [ ] Both keyboard (← → A D) and touch inputs route to `engine.moveBoat()`

# Story 1.7 — Game Over & Restart

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want the game to end on three misses and allow me to restart so that I can try again.

## Acceptance Criteria
- [ ] Third miss sets engine phase to 'game_over'
- [ ] Engine tick is a no-op in 'game_over' phase (physics frozen)
- [ ] `engine.restart()` resets: score=0, misses=0, parachutists=[], phase='attract'
- [ ] Helicopter and shark state also reset on restart
- [ ] Game Over sound plays on phase transition

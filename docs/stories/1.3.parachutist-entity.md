# Story 1.3 — Parachutist Entity

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want parachutists to fall at realistic speeds through three zones so that I have time to position my boat.

## Acceptance Criteria
- [ ] Parachutist renders in correct zone sprite per current zone (0, 1, 2)
- [ ] Fall speed is per-zone: slower in zones 0–1, faster in zone 2
- [ ] X position is locked from spawn — never changes mid-fall
- [ ] Maximum simultaneous parachutists: 2 (Game A), 3 (Game B)
- [ ] Game B fall rates are ~1.3× faster than Game A

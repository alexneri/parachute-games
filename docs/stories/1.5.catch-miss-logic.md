# Story 1.5 — Catch & Miss Logic

**Epic**: 1 — Core Game Engine  
**Status**: Draft  
**Priority**: P0

## User Story
As a player, I want the game to detect when I catch or miss a parachutist so that score and miss counter update correctly.

## Acceptance Criteria
- [ ] Catch fires when parachutist.xPosition === boat.position in zone 2 at subZoneProgress ≥ 0.85
- [ ] Miss fires when parachutist exits zone 2 with no position match
- [ ] Score increments by 1 on catch (capped at 999)
- [ ] Misses increments by 1 on miss
- [ ] missFlashMs set to MISS_FLASH_MS on miss
- [ ] shark.visible = true, shark.xPosition = parachutist X on miss
- [ ] Caught parachutist state → 'caught', missed → 'missed'

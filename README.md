# Parachute Game

> A pixel-faithful web recreation of the 1981 Nintendo Game & Watch Parachute (PR-21), playable in any browser.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

## Overview

This project recreates the classic Nintendo Game & Watch Parachute handheld game with 1:1 fidelity — identical LCD aesthetic, original timing, authentic mechanics, and synthesized sound effects. Zero install. Open in browser. Play instantly.

**Live**: https://parachute.vercel.app

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript 5.3
- **Rendering**: HTML5 Canvas API
- **Audio**: Web Audio API
- **Styling**: Tailwind CSS 3.4
- **Deployment**: Vercel
